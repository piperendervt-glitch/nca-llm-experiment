# Dataset: world_consistency

100 tasks for detecting logical contradictions between a world rule and a statement.

## Format

- `world_consistency_tasks.json` — JSON array
- `world_consistency_tasks.jsonl` — JSONL (one task per line)

## Fields

| Field | Type | Description |
|-------|------|-------------|
| task_id | int | Task index (0-99) |
| question | str | Statement to evaluate |
| label | bool | True=CONSISTENT, False=CONTRADICTION |
| world_rule | str | The world rule to check against |

## Statistics

- Total tasks: 100
- CONSISTENT: 50
- CONTRADICTION: 50

## Usage

```python
import json
with open("world_consistency_tasks.json") as f:
    tasks = json.load(f)
```
